# react-javascript-boilerplate
Simple Search to Github api with React, Webpack, Babel.

## Installation
```
$ git clone git@github.com:oeduardoal/react-javascript-boilerplate.git project
```

## Starting
### Install packages
```
$ cd project && npm i
```

## NPM
```
$ npm start
```
